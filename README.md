# 🏡 BlockVillage

3D воксельная песочница для Android в стиле Minecraft. Строй свой дом, сад, деревню!

## 🎮 Геймплей
- **Тап** — поставить блок
- **Долгий тап** — сломать блок
- **Драг** — вращать камеру
- **Два пальца** — зум + перемещение

## 🧱 Блоки
Трава, Земля, Камень, Дерево, Листва, Доски, Песок, Вода, Кирпич, Стекло, Цветы

## 🔧 Технологии
- Kotlin + OpenGL ES 2.0
- Воксельный чанковый мир 64×64
- Рейкастинг для точного попадания по блокам
- Автосохранение мира

## 🚀 Как собрать APK

1. Форкни/загрузи этот репозиторий на GitHub
2. Перейди во вкладку **Actions**
3. Нажми **Build APK** → **Run workflow**
4. Скачай APK из артефактов

Или собери локально:
```
./gradlew assembleDebug
```
APK будет в `app/build/outputs/apk/debug/app-debug.apk`

## 📱 Системные требования
- Android 7.0+ (API 24)
- OpenGL ES 2.0
- ~15 MB свободного места
